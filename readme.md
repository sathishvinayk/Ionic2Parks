### Installation
  Need to Have the below versions
    Typescript->2.3.2
    Node->6.10.2
    Npm->4.6.1
    googlemaps->3.26.11 

```
npm install --save @types/googlemaps 
```
```
npm install @types/googlemaps --save-dev --save-exact
```
### Import Statements

```
import {} from '@types/googlemaps'
```
